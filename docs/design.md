# Teaching Feeling 風 育成シミュレーション - 詳細設計書

## 概要

傷ついた過去を持つ少女を引き取り、日常生活を通じて関係性を築いていく純愛シミュレーション。

---

## コンセプト

| 項目 | 内容 |
|------|------|
| ジャンル | 関係構築シミュレーション |
| 設定 | 主人公（プレイヤー）が少女を引き取り、共に暮らす |
| 期間 | 明確な期限なし（日常の繰り返し） |
| 目標 | 少女の心を開かせ、信頼関係を築く |

---

## ステータス設計

### コアステータス

| 名前 | 説明 | 初期値 | 上限 |
|------|------|--------|------|
| 親密度 (affection) | 心の距離。会話・プレゼントで上昇 | 0 | 100 |
| 信頼度 (trust) | 安心感。一貫した優しさで上昇 | 0 | 100 |
| 心理状態 (mood) | 現在の気分。行動・イベントで変動 | 50 | 100 |
| 体調 (health) | 体の調子。低いと行動制限 | 80 | 100 |

### 隠しステータス

| 名前 | 説明 |
|------|------|
| 依存度 (dependence) | 主人公への依存。高すぎると特定エンド |
| 自立度 (independence) | 自己肯定感。成長イベントで上昇 |
| 恐怖 (fear) | 過去のトラウマ。特定行動で減少 |

### プレイヤーリソース

| 名前 | 説明 |
|------|------|
| 所持金 (money) | 外出・プレゼントに使用 |
| 日数 (days) | 経過日数（参考値） |

## アクションシステム

### アクション一覧

| アクション | 効果 | 解放条件 |
|------------|------|----------|
| 話す | 親密度+、会話イベント発生 | 最初から |
| なでる | 信頼度+（低信頼時は逆効果） | 最初から |
| 外出する | イベント、親密度+ | 信頼度 20+ |
| プレゼント | 親密度++、特定アイテムで大幅上昇 | 最初から |
| 着替え | 衣装変更、立ち絵反映 | 衣装所持時 |
| 休む | 日付進行、体調回復 | 最初から |

### アクション詳細

#### 話す
- 複数の会話トピックからランダム選択
- 親密度に応じて会話内容が変化
- 選択肢あり → 好感度への影響

#### なでる
- 頭 / 肩 / 手 などの部位選択
- 信頼度が低いと怯える（逆効果）
- 信頼度が高いと喜ぶ

#### 外出する
- ロケーション選択（後述）
- お金を消費
- イベント発生

#### プレゼント
- 所持アイテムから選択
- アイテムごとに効果が異なる
- 好みに合うと大幅上昇

## イベントシステム

### イベントタイプ

| タイプ | トリガー | 例 |
|--------|----------|-----|
| 会話イベント | 「話す」選択時 | 日常会話、相談 |
| タッチイベント | 「なでる」選択時 | 反応差分 |
| 外出イベント | 外出先で発生 | デート、買い物 |
| ランダムイベント | 日付進行時 | サプライズ、トラブル |
| 閾値イベント | ステータス条件達成 | 関係進展イベント |
| 季節イベント | 特定日付 | クリスマス、誕生日 |

### 閾値イベント（マイルストーン）

| 条件 | イベント |
|------|----------|
| 親密度 20 | 初めての笑顔 |
| 親密度 40 | 名前を呼んでくれる |
| 信頼度 30 | 過去を少し話す |
| 親密度 60 | 手を繋ぐ |
| 信頼度 50 | トラウマの告白 |
| 親密度 80 | 告白イベント |
| 親密度 100 + 信頼度 80 | トゥルーエンド解放 |

## 会話システム

### 構造

```typescript
interface Dialogue {
  id: string
  speaker: 'player' | 'girl' | 'narration'
  text: string
  expression?: Expression  // 表情
  choices?: Choice[]       // 選択肢（あれば）
  next?: string            // 次のダイアログID
}

interface Choice {
  text: string
  effects: StatusEffects
  next: string
}
```

### 表情タイプ

| 表情 | 使用場面 |
|------|----------|
| neutral | 通常 |
| happy | 嬉しい時 |
| sad | 悲しい時 |
| embarrassed | 照れ |
| scared | 怯え（低信頼時） |
| angry | 怒り |
| surprised | 驚き |
| loving | 愛情表現（高親密度） |

## データ構造

### GameState

```typescript
interface GameState {
  // 少女
  girl: {
    name: string
    stats: {
      affection: number      // 親密度
      trust: number          // 信頼度
      mood: number           // 心理状態
      health: number         // 体調
      dependence: number     // 依存度（隠し）
      independence: number   // 自立度（隠し）
      fear: number           // 恐怖（隠し）
    }
    currentOutfit: string    // 現在の衣装ID
    unlockedOutfits: string[] // 所持衣装
  }

  // プレイヤー
  player: {
    money: number
    items: Item[]
  }

  // 時間
  date: {
    day: number
    month: number
    year: number
    dayOfWeek: number
  }

  // 進行
  flags: Record<string, boolean>  // イベントフラグ
  seenEvents: string[]            // 既読イベント
  unlockedLocations: string[]     // 解放済みロケーション
}
```

### Phaser vs React のみ

Teaching Feeling 風であれば、**Phaser は不要**。
- メニューベースの操作
- 立ち絵 + テキスト表示
- React + CSS アニメーションで十分

**Phaser を削除して React のみに簡略化**することを推奨。
